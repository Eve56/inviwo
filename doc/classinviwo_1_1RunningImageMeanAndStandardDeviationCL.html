Detailed Description

Computes per-pixel standard deviation and mean value. Can be used both with external data and by letting the class manage the temporary and current data. http://en.wikipedia.org/wiki/Standard_deviation

Computes mean and standard deviation of given new values and iteration number.

Computes mean and standard deviation of all channels (RBGA) separately. This class manages mean and standard deviation layers. Output pointers are only valid if true is returned.
