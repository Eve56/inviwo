Detailed Description

Matches inviwo data formats to OpenCL image formats.

1, 2 and 4 channel data are supported by OpenCL. One channel data will be possible to retrieve using the x component. Two channel data will be possible to retrieve using the xy component. Four channel data will be possible to retrieve using the xyzw component.
